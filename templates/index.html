Your data file must contain at least two primary columns for forecasting:

  • Date/Time Index: Auto-detects columns like "Date", "Month", "Period", etc.
  • Sales Value: Auto-detects numeric columns like "Sales", "Revenue", "Amount", etc.
  • Category Filter (Optional): Automatically extracts categories if a "Product Category" or similar column is detected.
